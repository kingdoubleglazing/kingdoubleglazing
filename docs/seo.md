## Technical SEO infrastructure

### generateMetadata helper
Takes page-specific data, emits: title, description, canonical, OpenGraph, Twitter Card with defaults. Located at `lib/seo/generateMetadata.ts`.

### Schema helpers (all typed, JSON-LD)
Located at `lib/seo/schema/`:
- `localBusiness.ts` — full NAP, hours, geo, rating, services, sameAs
- `service.ts` — serviceType, areaServed, offers with priceRange
- `faqPage.ts` — auto-generated by FAQ component
- `article.ts` — headline, datePublished, dateModified, author, image
- `breadcrumbList.ts` — auto-generated from route

### Sitemap
`app/sitemap.ts` — reads route tree + data files automatically. No manual maintenance.

### Robots
`app/robots.ts` — allow all, disallow `/dev/`, point to sitemap.

### OG images
`next/og` for dynamic branded social previews per page. Fallback at `app/opengraph-image.tsx`.

### Canonical discipline
- Service pages: self-canonical
- Suburb pages: self-canonical when content is unique, parent service canonical when substantially shared
